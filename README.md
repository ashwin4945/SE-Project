This project is part of the prescribed curriculum for Software Engineering(SE) Course for BE 4th Sem, Dept of IS&E, NIE Mysore.

This Project is Website on Blood Bank Management System. It helps to store the data related to availabily of blood, Details of Donors etc..
The techstacks used are HTML, CSS and JS for Frontend.

Instructions to Run the Project
1. Install VS Code
2. Pull the project from this repository 
3. The main page is home.html. Run this prohject from that page.
